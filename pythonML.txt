## To know the version of python installed in your system
`$ python --version `

## Install python package to install machine learning libraries
`$ sudo apt-get -y install python3-pip`

## some data visualisation tools
`$ sudo apt-get install python3-numpy python3-scipy python3-matplotlib python3-pandas`

## some package is missing in default so install contrib
`$ pip3 install opencv-contrib-python`

`$ sudo apt install python3-opencv`

## If you are behind proxy use
`$ pip3 --proxy http://username:password@proxyhostname:port install packageName`

## If you getting some user permission issue
`$ pip3 install --user packageName`

## Prefered IDE 
`$ sudo apt install jupyter-notebook`
